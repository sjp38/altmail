altmail
=======

Find alternative mail address(es) for a user or members of a mailing list
group.


Usage
=====

```
$ altmail.py <user name or mailing list name>
```

For example,

```
$ altmail.py alice
alice@gmail.com
$
$ altmail.py members
alice@gmail.com
chatshire@gmail.com
david@yahoo.com
$
```
